# JSXRules 🚀

JSXRules is a React application built using Vite to demonstrate core JSX concepts, component structure, and modern UI styling with smooth transitions and animations. The project focuses on understanding JSX rules while presenting them in an interactive neon-themed interface with gradient text effects and a rotating logo animation.

Tech Stack: React, Vite, JavaScript, CSS (Transitions & Animations)

Project Structure:

JSXRules/
- public/
- src/
  - assets/
  - components/
  - App.jsx
  - main.jsx
  - index.css
- index.html
- package.json
- vite.config.js

Component Tree:

main.jsx  
└── App.jsx  
    ├── Navbar (inside components)  
    ├── Card / Content Section  
    ├── JSX Rules List  
    └── Footer  

Features:

- Component-based architecture  
- Organized folder structure  
- Smooth CSS transitions  
- Rotating animated logo  
- Gradient text styling  
- Hover effects and UI animations  
- Clean neon-inspired modern design  

Getting Started:

1. Navigate to the project folder:
   cd JSXRules

2. Install dependencies:
   npm install

3. Run the development server:
   npm run dev

The application runs at:
https://jsxruless.netlify.app/

Purpose:

This project was built to practice JSX syntax rules, component creation, folder structuring, styling enhancements, animations, and modern frontend development using React with Vite.
